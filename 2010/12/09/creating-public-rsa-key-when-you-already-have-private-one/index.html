Creating a public rsa key when you already have a private one

Posted on 2010, Dec 09 by Dima Moroz One min read

To create a public rsa key using a private one, simply type:

$ ssh-keygen -y -f /path/to/private_rsa > /path/to/public_rsa

Note that it is not adding user@hostname, so if you need it, add this in the end of public_rsa file with a leading space symbol
